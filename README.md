#SQL Practice Exercises

## Overview
This repository contains SQL practice exercises hand writen. The work focuses on core SQL concepts used in data analytics, including filtering, sorting, aggregate functions, grouping, CASE statements, joins, UNION, and query-based business analysis.

## Topics Covered
- SELECT statements
- DISTINCT
- WHERE filtering
- AND, OR, NOT, IN
- ORDER BY
- LIMIT
- COUNT, SUM, AVG, MIN, MAX
- GROUP BY
- HAVING
- CASE statements
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- UNION
- UNION ALL

## Exercise Files
### 1. SQL Fundamentals: SELECT & Filtering
Covers:
- SELECT

- LIMIT
- WHERE

- Combined conditions

### 2. Aggregate Functions & Grouping
Covers:
- COUNT()

- MIN() and MAX()
- GROUP BY
- HAVING
- ORDER BY with aggregates

### 3. SQL CASE Statements
Covers:
- Price categorization
- Order value labels
- Employee role classification
- Student grades
- Delivery performance
- Ticket priority labels
- Attendance status
- Stock status
- Class size category
- Discount eligibility

### 4. Joins, UNION, Filtering, and Aggregates
Covers:
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- UNION
- UNION ALL
- Filtering by salary, department, city, and project status
- GROUP BY and HAVING with employee and project data
